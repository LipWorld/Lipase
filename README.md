```
开机
  │
  ▼
Linux 启动 systemd
  │
  ├──► 启动 ollama.service（后台，等待 LLM 调用）
  ├──► 启动 whisper.service（后台，等待语音输入）
  └──► 启动 lipas.service（主进程）
              │
              ▼
         Flutter 启动
              │
              ├── 检查 ollama 连通性
              ├── 触发 ModelManager.ensureModels() ── 后台下载缺失模型（不阻塞 UI）
              ├── 加载用户记忆（Hive 本地数据库）
              ├── 初始化 lipa 解释器
              └── 显示主界面，等待输入
```
